Introducing lan-learn

We built lan-learn when we couldn't find open source learning software that fit our specific educational situation. Now we're releasing our very work-in-progress alpha version with the hope that like-minded devs will pick it up and start customizing it for their own needs.

Simple. Expandable. Flat-file.

This means new features are as easy as writing new ones. Want to highlight key terms and link them to wikipedia? It's a plugin. Quizzes with automatic grading? Another plugin.
